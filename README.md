# Coloured_linegame
Colored Lines is a Python-based puzzle game using Tkinter. It is played on a 9×9 grid where the player moves colored balls to form straight lines of five or more matching balls. Formed lines are removed and scored, while new random balls appear. The game ends when the grid is full.
9×9 interactive game board

Movement and placement of colored balls using mouse clicks

Automatic detection of straight-line formations (≥5 balls)

Score calculation based on number of balls removed

Random ball generation for continuous gameplay

Graphical UI created using Tkinter

Image loading and display using PIL (Pillow)
